shape_layout 的后果：
- `SizeStrategy::Fill` + `max_width` → 自动缩放
- `VAlign::Baseline` → 自动对齐
- `LayoutWarning::ElementTooWide` → 明确告诉你"这个名字太长，模板需要调整"

```
shape_layout：

- 1000 次生成，1000 次结果完全一致
```

**人眼是"模拟的"，shape_layout 是"数字的"。** 在工业制造中，模拟信号无法驱动数控机床。

### 4. 跨设备套准的"死亡精度"

```
工业制造的典型流程：
1. 印刷机：在亚克力上印出彩色图案（精度 ±0.1mm）
2. 切割机：沿着切割线切出外形（精度 ±0.05mm）
3. 雕刻机：在指定位置刻字（精度 ±0.02mm）

三个设备必须"套准"（Registration）：
- 印刷的图案和切割线必须对齐
- 雕刻的文字必须在印刷的预留区域内

手工排版的套准：
- 第1个：人眼看着对齐了 → 实际误差 ±0.5mm → 废品率 5%
- 第100个：人眼疲劳 → 实际误差 ±1mm → 废品率 20%

shape_layout 的套准：
- 所有坐标基于同一个容器 BezPath 计算
- 印刷层、切割层、雕刻层共享同一套几何基准
- 误差仅来自机器本身（±0.05mm），与排版无关
- 废品率 < 0.1%
```

**在工业制造中，"好看"不是标准，"可重复、可预测、可上机"才是标准。**
